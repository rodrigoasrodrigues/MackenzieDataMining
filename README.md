# Mackenzie Data Mining
Esse repositório contem uma análise de diferentes métodos para 
resolução do desafio __New York City Taxi Trip Duration__ disponóvel
no Kaggle: https://www.kaggle.com/c/nyc-taxi-trip-duration.

Essa análise é apresentada como trabalho de conclusão da matéria de 
__Data Mining__ do curso __Big Data Analytics__ da __Universidade 
Presbiteriana Mackenzie__.

___

Solução por Bruno Matrone e Rodrigo Rodrigues.
